Movie API Documentation Page

Following are the the URL endpoints and the corresponding request methods associated with the key features of the Movie Application.
Request URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Get a list of movies /allMovies GET None None A JSON object holding data about all the movies 
 
                Structured like:
                {
                    movieId: '01',
                    movieTitle: 'The Godfather',
                    movieGenres: ['Epic', 'Gangster', 'Tragedy', 'Crime', 'Drama'],
                    movieDirector:
                    {
                        firstName: 'Francis Ford',
                        lastName: 'Coppola',
                        birthYear: '1939',
                        birthPlace: 'USA',
                        deathYear: '1986'
            
                    }
                }
Get a single movie data by movie title /allMovies/:movieTitle GET movieTitle None A JSON object holding data about a single movies 
 
                Output:
                {
                    movieId: '04',
                    movieTitle: 'Jaws',
                    movieGenres: ['Sea Adventure', 'Survival', 'Adventure', 'Drama', 'Thriller'],
                    movieDirector:
                    {
                        firstName: 'Steven',
                        lastName: 'Spielberg',
                        birthYear: '1946',
                        birthPlace: 'USA',
                        deathYear: null
                    }
                }
Get movie genres by title /allMovies/:movieGenre GET movieGenre None A JSON object holding data about a movie genres 
 
                Output:
                {
                    'Sea Adventure', 
                    'Survival', 
                    'Adventure', 
                    'Drama', 
                    'Thriller'
                }
Get movie director data by title /allMovies/:movieDirector GET movieDirector None A JSON object holding data about a movie director 
 
                Output:
                movieDirector:
                {
                    firstName: 'James',
                    lastName: 'Wan',
                    birthYear: '1977',
                    birthPlace: 'Malaysia',
                    deathYear: null
                }
Register a new user /user POST username None A JSON object holding data a user data 
 
                Structured like:
                {
                    username: 'John',
                    password: 'John@123',
                    email: 'John.Camell@gmail.com',
                    dateOfBirth: '07/08/1990',
                    favoriteMovies: ['Jurassic Park]
                }
Update user information /user/:userName PUT userName A JSON object holding updated user data (userName) 
 
                {
                    username: 'John',
                    password: 'John@123',
                    email: 'John.Camell@gmail.com',
                    dateOfBirth: '07/08/1990',
                    favoriteMovies: ['Jurassic Park']
                }
Success or Failure Message
Add a movie to user favorite list /user/:userName/:favoriteMovies POST userName and movieTitle A JSON object holding a movie title 
 
                {
                    username: 'John',
                    password: 'John@123',
                    email: 'John.Camell@gmail.com',
                    dateOfBirth: '07/08/1990',
                    favoriteMovies: ['Jurassic Park']
                }
A JSON object holding user updated favorite movies list 
 
                {
                    favoriteMovies: ['Jurassic Park','Jaws']
                }
Delete a movie to user favorite list /user/:userName/:favoriteMovies DELETE userName and movieTitle None A text message or updated favorite list
Deregistering a user /user/:userName DELETE userName None Confirmation or Error Message
